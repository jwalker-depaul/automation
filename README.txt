***   SETUP   ***
1. Download the latest chromedriver, ensure the driver supports the correct version of the Chrome browser
2. Download the latest selenium libraries.
3. In the project, make sure to reference the selenium libraries, include all Jars, even in the lib folder
4. Ensure JDK is installed
